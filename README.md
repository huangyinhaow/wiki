# Bruce Wiki

My personal Wiki for recording Web development knowledge.

## Tech Stack

- React
- TypeScript
- Docusaurus

## Usage

### Install

```bash
pnpm i
```

### Start

```bash
pnpm dev
```

### Build

```bash
pnpm build
```
